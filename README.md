# Multi Find & Replace

Vibe coded multi find and replace web extension.

## Features
- Remembering multiple find and replace patterns
- Undo
- Regex
- Case sensitivity

## Installation

1. Clone the repo
   ```bash
   git clone https://github.com/YOUR_USERNAME/multi-find-replace.git
   ```

2. Open Chrome and go to `chrome://extensions/`

3. Enable **Developer mode** (toggle in top right)

4. Click **Load unpacked**

5. Select the cloned folder

Done! The extension icon will appear in your toolbar.
